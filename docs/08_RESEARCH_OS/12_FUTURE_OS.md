# Purpose

Knowledge State is one of the fundamental abstractions inside AIVE.

A Knowledge State represents the complete understanding of a workspace at a specific point in time.

Everything evolves from one Knowledge State into another.

---

# Philosophy

Knowledge is never static.

Only its current state is observed.

Discovery changes state.

Validation strengthens state.

Learning evolves state.
