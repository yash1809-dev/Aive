# Purpose

This document defines Discovery as a persistent object.

Discoveries are not reports.

They are evolving knowledge entities.

---

# Definition

A Discovery Object represents a validated understanding that emerged through evidence-backed reasoning.

Unlike generated text,

Discovery Objects continuously evolve.

---

# Components

Every Discovery Object contains:

Discovery ID

Title

Summary

Reasoning References

Evidence References

Claim References

Confidence

Novelty

Impact

Validation Status

Version

History

Contributors

Related Opportunities

# Discovery Evolution

Discoveries should never be frozen.

Every new evidence object may improve or invalidate them.

# Goal

A Discovery Object represents persistent understanding rather than generated output.
