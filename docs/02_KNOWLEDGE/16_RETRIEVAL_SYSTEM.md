# Purpose

Retrieval provides relevant knowledge to the Discovery Engine.

Retrieval is not the objective.

Retrieval is infrastructure supporting reasoning.

---

# Philosophy

Traditional RAG retrieves documents.

AIVE retrieves Knowledge Objects.

This distinction dramatically improves reasoning quality.
