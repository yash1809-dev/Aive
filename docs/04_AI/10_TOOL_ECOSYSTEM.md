# Purpose

Artificial Intelligence alone is insufficient for discovery.

Agents must interact with external tools capable of searching, calculating, simulating, executing and validating knowledge.

This document defines the Tool Ecosystem of AIVE.

# Philosophy

The combination produces intelligent execution.
