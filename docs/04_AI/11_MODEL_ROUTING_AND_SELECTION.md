# Purpose

This document defines how AIVE selects the most appropriate model for every cognitive task.

Model selection should be dynamic, measurable and continuously optimized.

---

# Philosophy

Choose the best model for the task.

Not the biggest model.

Not the newest model.

Not the cheapest model.
