# Purpose

Agents require memory to operate effectively.

Agent Memory defines what an individual agent knows during execution and how that knowledge interacts with the global Memory Engine.

---

# Philosophy

Agents should remember only what is necessary.

Long-term intelligence belongs to the Memory Engine.
