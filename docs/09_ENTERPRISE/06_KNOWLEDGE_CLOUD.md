# Purpose

Every organization possesses enormous knowledge distributed across thousands of systems.

The Enterprise Knowledge Cloud unifies this fragmented knowledge into one continuously evolving organizational intelligence.

---

# Philosophy

Organizations should never lose knowledge because it exists in different systems.

Knowledge should exist independently of applications.

Applications change.

Knowledge remains.
