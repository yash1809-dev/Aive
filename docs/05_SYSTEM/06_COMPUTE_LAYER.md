# Purpose

The Compute Layer provides computational resources for executing capabilities, agents, models and tools.

It abstracts physical hardware from cognitive execution.

---

# Philosophy

Execution should be location independent.

The same capability may execute on:

Laptop

↓

Workstation

↓

Private Cloud

↓

Public Cloud

↓

GPU Cluster

↓

Distributed Infrastructure

without changing system behavior.
