# Purpose

This document defines the product vision of AIVE.

Everything below this layer exists for one purpose:

Helping humans make better discoveries.

The Product Layer transforms AIVE's cognitive architecture into an experience that researchers, scientists, startups, enterprises and governments can use every day.
