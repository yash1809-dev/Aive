# Purpose

The Knowledge Explorer allows users to navigate every piece of knowledge inside a workspace.

Unlike traditional document browsers, the Knowledge Explorer visualizes relationships, evidence and semantic connections rather than folders.

Knowledge should be explored rather than searched.

---

# Philosophy

Documents hide knowledge.

Knowledge Objects reveal it.

The explorer should expose understanding instead of storage.
