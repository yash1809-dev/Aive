# Purpose

The Novelty Engine attempts to disprove every proposed discovery before it can be accepted.

Its objective is not to prove novelty.

Its objective is to reject false novelty.

This adversarial philosophy is one of AIVE's primary differentiators.

---

# Responsibilities

Search prior art.

Search semantic equivalents.

Search patents.

Search publications.

Search startups.

Search products.

Estimate novelty.

Reject duplicates.

---

# Non-Responsibilities

Generate discoveries.

Validate evidence.

Perform criticism.

Generate reports.
