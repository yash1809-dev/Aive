8. Evidence-Centred Design instrument using staged in-class polls as designed evidence probes
9. Flexible multidimensional item-response model with answer choices having different directions in the multidimensional space of student knowledge
10. Stage-wise evaluation framework with independent evaluation of each stage of the solution to assess intermediate reasoning processes.
11. Knowledge graph-driven visualization and multimodal interaction utilizing large language models, artificial intelligence-generated illustrations, three-dimensional meridian-acupoint models, and evidence-based literature.
12. Prerequisite Knowledge Graph (PKG) construction, Lightweight Policy Gradient Method (PPO) for sequencing decision-making, Large Language Model (LLM) for Socratic exchange
13. Symbolic machine learning, classical machine learning, experimental design with multiple protocols and datasets
14. Constrained adaptive AI Interviewer with decomposed process reward mechanism, providing non-directive knowledge and agency scaffolds in multi-turn interaction
15. Soft Knowledge-Graph Priors for Bayesian Network Structure Learning from Sparse Discrete Data using a weighted directed knowledge graph and a logit-form prior
16. A semantic analysis pipeline was developed using definitions and scale items extracted from over 14,000 publications to investigate how researchers actually used learner agency and autonomy.
17. Component exposition, a paradigm that allows authors to embed selected workflow components into arbitrary web contexts
18. Distractor-aware architecture utilizing text input encoding, order-aware concatenation, and order-invariant summation to isolate the contribution of distractor content and order
19. Sequential fine-tuning approach utilizing LoRA (Low-Rank Adaptation) with 4-bit quantization for efficient model training
20. Generative AI systems, specifically large language models (LLMs), are used as surrogates for human deliberation in various domains.
