## Step 7: Generate the Paper

1. Click the big blue button: **"⚡ Generate Research Paper"**

2. **Status message appears**: "Generating research paper... This may take 1-2 minutes."

3. **What's happening behind the scenes**:
   - Backend gathers evidence from all 4 linked nodes
   - Extracts source documents for citations
   - Builds grounded context
   - LLM generates each section (Abstract → Introduction → ... → Conclusion)
   - Assembles markdown with references
   - Saves to `reports/` directory

4. **Wait patiently** - takes 60-120 seconds depending on:
   - Number of linked nodes
   - Number of sections
   - LLM response time

5. **Success message appears**:
   ```
   ✓ Paper generated successfully!
   File: research_paper_[title]_[timestamp].md
   Linked: 4 nodes | Evidence: X items | Citations: Y
   ```

6. **Auto-navigation**: Page automatically switches to **Reports tab** after 2 seconds

## Step 9: Verify Paper Quality

Check these key aspects:

### ✅ Evidence Citations
- Look for `[item_id]` format citations throughout
- Example: "Recent research [paper_ml_2024] demonstrates..."
- Should appear in every section

### ✅ References Section
- At the bottom of the paper
- Lists all cited sources
- Includes titles, years, URLs

### ✅ No Hallucination
- All facts should trace back to your linked nodes
- Claims should reference evidence
- If insufficient evidence, should state "further research needed"

### ✅ Coherent Structure
- Each section flows logically
- Abstract summarizes entire paper
- Introduction sets up the problem
- Methods/Results present findings
- Discussion analyzes implications
- Conclusion wraps up

## Common Issues & Solutions

### Issue: "No knowledge nodes available"
**Solution**: Run the pipeline first
```bash
# In browser, click "Run Pipeline" button
# Wait for completion
# Then try Paper Builder again
```

### Issue: "Please link at least one knowledge node"
**Solution**: You forgot to click + buttons! Link 3-5 nodes first.

### Issue: Paper generation takes >3 minutes
**Solution**: 
- Check Ollama is running: `ollama list`
- Check terminal for LLM errors
- Reduce number of linked nodes (try 3-4 instead of 10)

### Issue: Generated paper has no citations
**Solution**: 
- Linked nodes may lack evidence fields
- Link opportunities instead (they have rich evidence)
- Re-run pipeline to populate evidence

### Issue: Paper content is generic
**Solution**:
- Link more specific nodes
- Choose nodes with detailed problem/technology fields
- Add discoveries for richer insights
